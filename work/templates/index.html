แนะนำระบบ

ระบบเป็นระบบสำหรับแจ้งเตือนการหลับในเพื่อให้ตื่นตัวอยู่เสมอขณะขับขี่รถยนต์

วิธีใช้งาน

1.กดไปที่เริ่มการทำงานการตรวจจับ ที่อยู่ในหน้าการทำงาน

2.กด Start Detection แล้วรอสักครู่ ระบบจะเริ่มเปิดกล้องและทำงานระบบ

ปัญหาของระบบ

ในระบบยังคงมีปัญหาอยู่

1.ระบบไม่สารมารถกดปิดได้ ถ้าจะปิดการทำงานของระบบต้องปิดการทำงานใน command lind โดยตรงเลย ซึ่งอาจจะทำให้เว็บไซต์หยุดการทำงาน และต้องดำเนินการทำงานใหม่

2.กล้องของระบบยังใช้เปิดของกล้องที่เป็นโฮสต์หลัก ไม่สามารถเปิดให้ผู้ใช้อื่นเข้ามาในเว็บไซต์แล้วเปิดเป็นกล้องของตัวเองได้
